## Features
📓 "Home" Page
<br>🔵 "Generate Week" Button: Generates 7 days of 3 meals from database data (not AI)
<br>🔵 "Generate Day" Button: Generates one day of 3 meals from database data (also not AI)
<br>🔵 "Pin" Button: Pins a meal card or food card
<br>🔵 "Shuffle Button": Generates new food card or meal card to replace any non-pinned cards
<br>
<br>📓 "Update Database" Page
<br>🔵 CRUD Operations
